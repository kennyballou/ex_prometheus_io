# Elixir Prometheus.IO #

Elixir API library for querying data from [Prometheus][1].

## Installation ##

The package can be installed by adding `ex_prometheus_io` to your list of
dependencies in `mix.exs`.

    def deps do
      [{:ex_prometheus_io, "~> 0.0.1"}]
    end
